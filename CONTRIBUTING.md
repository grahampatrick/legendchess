# Contributing

Thanks for wanting to help! This file will grow; for now:

- Read [plan.md](plan.md) first — every PR is reviewed against its Architectural
  Non-Negotiables.
- `pnpm lint && pnpm typecheck && pnpm test` must be green.
- Non-obvious decisions get an ADR in `docs/adr/` in the same PR.
- **Adding a famous game** (the most valuable contribution!) will be a config-file PR once the
  forge pipeline lands (Milestone M2) — a guide will appear here then.

By contributing you agree your work is licensed under GPL-3.0-or-later.
